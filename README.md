# Power BI Projects Portfolio

Welcome to my **Power BI Projects** repository! This collection showcases interactive dashboards and data visualizations developed during my internship at **HBIC Solutions**. The projects span multiple domains, including sales, finance, healthcare, e-commerce, HR, and social analytics, demonstrating my skills in data modeling, visualization, and insights generation.

---

## 📂 Repository Structure

Each folder represents a distinct Power BI project and typically contains:

- `.pbix` files – Power BI dashboard files.
- `.csv` or other data sources – the underlying dataset used for the dashboard.
- Supporting documents – reports or notes (optional).

Example structure:
PowerBI-Projects/
├─ Adidas Sales/
│ ├─ adidas_sales.pbix
│ └─ data.csv
├─ Bank Deposit/
│ ├─ bank_deposit.pbix
│ └─ bank_data.csv
├─ Netflix Movies and TV Shows/
│ ├─ netflix_dashboard.pbix
│ └─ netflix_data.csv
├─ IPL/
│ ├─ ipl_analysis.pbix
│ └─ ipl_data.csv
